# Currency Converter in python

This is a simple Python project that converts currency values between different countries.

The program takes user input and converts the amount based on predefined exchange rates.

---

## Features

- Convert USD to INR
- Convert INR to USD
- Convert USD to EUR
- Convert EUR to USD
- Convert EUR to INR
- Convert INR to EUR
- Handles invalid input

---

## Technologies Used

- Python

---

## How It Works

1. User enters the amount
2. User selects source currency
3. User selects target currency
4. Program converts the value using fixed rates
5. Output is displayed in the terminal

---

## Example

Enter amount: 10  
Convert from: USD  
Convert to: INR  

Output: Amount in INR: 930  

---

## Note

- Currency exchange rates used in this project are fixed values.
- Real-world exchange rates change frequently.
- Example: 1 USD ≈ 83–93 INR (may vary over time).
- So the results may not be accurate in real-time.
